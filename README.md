# Deskripsi

- project aplikasi web sederhana untuk diagnosa penyakit berbasis AI
- api AI yang digunakan adalah groq

## Syarat & Ketentuan

1. Sudah terinstall python 3.x
2. Sudah menginstal library flask & groq
3. Mengerti konsep routing

## Install flask

- pip install flask

## install groq

- pip install groq

### Note

Disarankan untuk menggunakan Venv ( virtual environtment )
